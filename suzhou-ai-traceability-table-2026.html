🔗 ARAS PLM 导入步骤

  1. 导出数据：点击页面顶部"下载"按钮，选择 Excel (.xlsx) 格式下载追溯表数据文件。
  2. 数据准备：打开导出的 Excel 文件，检查需求ID、技能ID等必填字段是否完整，删除示例数据行（如有）。
  3. 登录ARAS：打开 ARAS Innovator 管理控制台，使用管理员账号登录。
  4. 导入向导：导航至 Administration → Import，选择"Excel Import Wizard"。
  5. 映射字段：在字段映射界面，将Excel列与ARAS Item类型属性逐一对应：
    需求ID → Item Number、技能ID → Classification、技术组件 → Description、状态 → State。
  6. 执行导入：确认映射无误后点击"Import"，等待系统完成数据写入。
  7. 验证结果：在ARAS中搜索已导入的Item，检查字段值是否正确，关联关系是否建立。

📋 Jira 导入步骤

  1. 导出数据：下载追溯表的 CSV (.csv) 格式文件。
  2. 格式适配：使用文本编辑器打开CSV文件，确保首行为Jira字段名映射行，格式如下：
    Summary,Description,Priority,Due Date,Assignee,Labels
  3. 创建项目：在Jira中创建项目（如 SCSAI-Trace），设置Issue类型为"Task"或"Epic"。
  4. 导入入口：导航至 项目设置 → 外部系统导入 → CSV。
  5. 映射字段：Jira导入向导中完成CSV列到Jira字段的映射：
    需求ID → Summary、技能ID → Labels、优先级 → Priority、计划上线 → Due Date、负责人 → Assignee。
  6. 批量创建：确认映射后点击"Begin Import"，Jira将批量创建Issue。
  7. 建立关联：导入完成后，使用Jira的"Link Issues"功能建立需求→技能→组件之间的关联关系。
