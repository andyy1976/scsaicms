# AI数字内容生成发布管理与获客系统 - 整合方案

## 一、产品定位

**SCSAI ContentOS** — AI数字内容生成发布管理与获客系统

> 一站式 AI 内容数字员工平台：自动采集 → AI评分 → 智能生成 → 多平台发布 → 精准获客

### 目标用户
| 用户类型 | 使用场景 | 付费模式 |
|---------|---------|---------|
| 个人/自媒体 | 热点追踪+AI写作+公众号发布 | 免费版/订阅制 |
| 中小企业 | 品牌内容+获客+私域运营 | 订阅制/年费 |
| 大型企业 | PLM集成+定制内容+全员培训 | 项目制+年度维护 |

### 四大产品线
1. **数字员工** — AI内容数字员工、营销数字员工、客服数字员工
2. **人工智能** — 大模型、智能体、AI应用、多模态
3. **智能制造** — PLM/MES/QMS/ERP 工业软件
4. **机器人** — 具身智能、人形机器人、工业机器人

## 三、数据流

### 3.1 内容入库流
```
微博/知乎/Reddit/宇树 → fetch-hot.js → AI评分(ai-scorer.cjs) 
    → cms-bridge.cjs → MySQL lvbo_article → CMS 自动展示
```

### 3.2 内容发布流
```
CMS文章 → content-pipeline → AI改写/适配 → 多平台发布
    → 微信公众号 (Playwright自动化)
    → 小红书 (API)
    → 抖音 (API)
    → 企业网站 (CMS直写)
```

### 3.3 API 集成
```
Node.js Pipeline ──POST──▶ /index.php?s=Contentapi/push
                         ──GET──▶ /index.php?s=Contentapi/categories
                         ──GET──▶ /index.php?s=Contentapi/health
```

## 六、部署方案

### 6.1 单机部署（开发/小规模）
```
Nginx → PHP-FPM (CMS)
      → Node.js (Content Pipeline + API)
      → MySQL (共享数据库)
```

### 6.2 生产部署
```
CDN → Nginx → PHP-FPM (CMS, 多实例)
              → Node.js API (Content Pipeline, PM2)
              → MySQL Master-Slave
              → Redis (缓存)
```
