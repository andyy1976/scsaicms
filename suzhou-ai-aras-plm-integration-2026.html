🔧 ARAS API版自动化流水线

将Jenkinsfile和GitLab CI脚本中的Jira API调用替换为 ARAS REST API 调用，其他流水线逻辑完全不变。

🔑 ARAS API认证配置

environment { // ARAS PLM配置 ARAS_URL = 'https://scsai-plm.yourcompany.com/innovator' ARAS_DATABASE = 'SCSAI' ARAS_USERNAME = 'automation-bot' ARAS_PASSWORD = credentials('aras-automation-password') // 平台配置 PLATFORM_API_URL = 'https://digital-employee.yourcompany.com/api/v1' DOCKER_REGISTRY = 'registry.yourcompany.com' DOCKER_NAMESPACE = 'digital-employee' }

📡 初始化阶段 — 从ARAS获取技能信息

stage('初始化与参数校验') { script { // 从ARAS获取技能信息 def arasAuth = "${ARAS_USERNAME}:${ARAS_PASSWORD}" .bytes.encodeBase64().toString() def response = httpRequest( url: "${ARAS_URL}/server/odata/DigitalEmployeeSkill('${params.SKILL_ID}')", headers: [[name: 'Authorization', value: "Basic ${arasAuth}"], [name: 'Accept', value: 'application/json']] ) def skillInfo = readJSON text: response.content env.SKILL_NAME = skillInfo.value[0].name env.CODE_MODULE_PATH = skillInfo.value[0].code_module_path env.CURRENT_VERSION = skillInfo.value[0].version env.ARAS_ITEM_ID = skillInfo.value[0].id // 更新ARAS中技能状态为"构建中" httpRequest( url: "${ARAS_URL}/server/odata/DigitalEmployeeSkill('${env.ARAS_ITEM_ID}')", method: 'PATCH', requestBody: '{"status":"构建中","build_url":"${BUILD_URL}"}' ) } }

✅ 完成阶段 — 更新ARAS与创建基线

stage('更新ARAS与完成') { script { // 1. 更新技能版本号和状态 httpRequest(method: 'PATCH', url: "...DigitalEmployeeSkill('${ARAS_ITEM_ID}')", requestBody: '{"version":"${NEW_VERSION}","status":"已上线"}') // 2. 创建部署记录Item httpRequest(method: 'POST', url: "...DeploymentRecord", requestBody: '{"skill_id":"...","version":"...","status":"成功"}') // 3. 添加完成评论 httpRequest(method: 'POST', url: "...Comments", requestBody: '🎉 技能全流程自动化完成') // 4. 自动创建版本基线 httpRequest(method: 'POST', url: "...CreateBaseline", requestBody: '技能v${NEW_VERSION}发布基线') } }
